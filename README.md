Teliot1 es un nodo LoRaWAN experimental y abierto para uso público para visualización de telemetría de forma gratuita y como muestra. Es configurable para añadir distintos sensores y actuadores remotos.Trabaja en la frecuencia de 868.1MHz y
y es facilmente instalable sin necesidad de infraestructuras ni conexión a internet.

Mas información en https://www.miprochip.com/2025/11/18/miprochip-teliot1/.
